Vehicle routing problem with backhauls (VRPB)

The subdirectory INSTANCES contains the following benchmark instances:

	G (62 instances)
	Randomly generated Euclidean instances
	Source:
	Goetschalckx M., Jacobs-Blecha C. 
	The vehicle routing problem with Backhauls. 
	Eur. J. Oper. Res., 42: 9-51 (1989)

	T (57 instances)
	eil (33 symmetric instances)
	ftv (24 asymmetric instances)
	Toth, P., Vigo, D.:
	An Exact Algorithm for the Vehicle Routing Problem with Backhauls.
	Transp. Sci., 32(4):372-385 (1997)

The subdirectory TOURS contains the best tours found by LKH-3.

Tabulated results can be found in the subdirectory RESULTS.
